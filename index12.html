Três formas de usar o CSS

Primeira forma: definindo na própria variável

É definido dentro da própria variável, chamado de "inline".
Normalmente usado quando precisamos editar somente aquela parte do código, sem afetar as demais.

    Exemplo: 
            < h2 style = "color: blue"; >

    Segunda forma: definindo no cabeçalho

    É difinido dentro da tag head, chamado de "internal".
    Normalmente é usado quando definimos um padrão para aquela tag em específico,
    para não ter muitas repetições desnecessárias dentro do código. Exemplo: 

            h2{
            color: plum;
        }

    Terceira forma: criando um arquivo de CSS

    É a forma mais usada, onde criamos um arquivo externo de CSS chamado de
    "style.css" e deixamos o HTML apenas para a estrutura. Exemplo: 

            < link rel="stylesheet" href="style.css">
