O que é semantica? e porque é impotante no HTML5?

A Semântica HTML é o uso correto das tags HTML
que indicam o significado do conteúdo de uma página web.
Basicamente significa que cada tag deve ser usada da forma certa para
descrever o que o conteúdo que ela representa, facilitando a compreensão,
manutenção e acessiblidade. 😃
